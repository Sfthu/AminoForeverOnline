<h1 align="center">
  AminoForeverOnline
</h1>

<p align="center">Forever online script for Amino</p>
<p align="center">Can be used on Heroku</p>


#### How to start?
1. Account must be in a community
2. Download online.py, requirements.txt and Procfile.
3. Run online.py
4. Enjoy
